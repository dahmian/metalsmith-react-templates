This is a demo of Isomorphic React using the static site generator Metalsmith.

When you first enter the site, you were served a full HTML page, but subsequent page loads use JSON. But unlike an typical Isomorphic app, all the HTML and JSON are generated in advance. This has a few interesting effects. Don't want to pay for server? Host your site on GitHub Pages. Enjoy the benefits of GitHub Pages CDN caching your static content and satisfy your need for speed. Not interested in GitHub Pages? Upload your site to your own web server and CDN and watch your resource usage plummet.
