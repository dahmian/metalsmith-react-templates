This is a demo of Isomorphic React. This HTML page was created with React in node, but it can also be rendered with client side code. This demo allows you to load another article through AJAX. Before you load the article, make sure to enter in some text in the input field.

If everything works, after you click the link below, the article will load and the input will still contain your text. If the input clears, that means that the node react and the client side react are out of sync.
